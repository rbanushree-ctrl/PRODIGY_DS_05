# Traffic Accident Data Analysis 🚦

## Overview
This project analyzes synthetic traffic accident data to identify patterns related to weather, road conditions, and time of day. It also visualizes accident severity and hotspot regions.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Faker

## Features
- Accident severity analysis
- Weather and road condition analysis
- Time-based accident pattern detection
- Hotspot visualization using coordinates

## Dataset
The dataset contains 5,000 synthetic traffic accident records with:
- Weather conditions
- Road conditions
- Severity levels
- Vehicle types
- GPS coordinates
- Casualty counts
